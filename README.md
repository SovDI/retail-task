# Тестовое задание: Интеграция RetailCRM -> Supabase -> Dashboard

## Описание проекта
Скрипт синхронизации заказов из RetailCRM в базу данных Supabase с уведомлением в Telegram о крупных заказах (> 50,000 ₸) и визуализацией данных на дашборде.

## Использованные технологии
* Python (requests, supabase-py)
* RetailCRM API v5
* Supabase (PostgreSQL)
* Telegram Bot API
* Chart.js (для дашборда)

## Как я решал задачу (для отчета)
1. **Промпты для ИИ:** Использовал подсказки для генерации базовой структуры скриптов на Python для работы с REST API и библиотекой Supabase.
2. **Трудности:** Столкнулся с ошибкой `[Errno 8] nodename nor servname provided`. 
3. **Решение:** Проблема была в неверном DNS-разрешении адреса Supabase. Решил заменой домена `.com` на `.co` в настройках и очисткой API-ключей от лишних символов.
